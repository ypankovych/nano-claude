**2. Create `nano_claude/panels/terminal.py`:**

Create a new file with the following implementation:

- **Imports:** `os`, `ComposeResult` from textual.app, `reactive` from textual.reactive, `Static` from textual.widgets, `ContentSwitcher` from textual.widgets, `BasePanel` from nano_claude.panels.base, `TerminalWidget` and `PtyExited` from nano_claude.terminal.widget, settings constants from nano_claude.config.settings.

- **Class `TerminalPanel(BasePanel)`:**
  - Class attribute `_tab_counter: int = 0` (instance-level, set in `__init__`)
  - Reactive `is_minimized = reactive(True)` (starts minimized per CONTEXT.md: hidden by default)

  - **`__init__(self, **kwargs)`:** Call `super().__init__(**kwargs)`, set `self._tab_counter = 0`.

  - **`DEFAULT_CSS`:**
    ```python
    DEFAULT_CSS = """
    TerminalPanel {
        height: auto;
    }
    TerminalPanel #terminal-tab-bar {
        height: 1;
        background: $surface;
        color: $text-muted;
        padding: 0 1;
    }
    TerminalPanel #terminal-status-line {
        height: 1;
        background: $surface;
        color: $text-muted;
        padding: 0 1;
    }
    """
    ```

  - **`compose(self) -> ComposeResult`:**
    - Set `self.panel_title = "Terminal"`
    - Yield `Static("", id="terminal-tab-bar")`
    - Yield `ContentSwitcher(id="terminal-switcher")`
    - Yield `Static(TERMINAL_STATUS_IDLE, id="terminal-status-line")`

  - **`on_mount(self)`:** Call `self._apply_minimized_state()` to set initial display states (minimized by default).

  - **`add_tab(self) -> None`:**
    - Increment `self._tab_counter`
    - Generate `tab_id = f"shell-{self._tab_counter}"`
    - Get shell: `shell = os.environ.get("SHELL", "/bin/sh")`
    - Check tab limit: `if len(list(switcher.children)) >= TERMINAL_MAX_TABS:` post a notification and return
    - Create `terminal = TerminalWidget(command=shell, id=tab_id)`
    - Get `switcher = self.query_one("#terminal-switcher", ContentSwitcher)`
    - Call `switcher.mount(terminal)`
    - Set `switcher.current = tab_id`
    - Call `self._update_tab_bar()`
    - Call `terminal.focus()`

  - **`close_active_tab(self) -> None`:**
    - Get `switcher = self.query_one("#terminal-switcher", ContentSwitcher)`
    - If `switcher.current is None`: return
    - Save `current_id = switcher.current`
    - Try to get the current terminal widget and call `terminal.stop_pty()` before removal
    - Get `remaining = [c for c in switcher.children if c.id != current_id]`
    - If remaining: set `switcher.current = remaining[-1].id`
    - Else: set `switcher.current = None`
    - Remove the terminal widget: `switcher.get_child_by_id(current_id).remove()`
    - Call `self._update_tab_bar()`
    - If not remaining: call `self.minimize()` (last tab closed -> minimize per CONTEXT.md)

  - **`minimize(self) -> None`:** Set `self.is_minimized = True`

  - **`restore(self) -> None`:**
    - Get `switcher = self.query_one("#terminal-switcher", ContentSwitcher)`
    - If `not list(switcher.children)`: call `self.add_tab()` first (ensure at least one tab)
    - Set `self.is_minimized = False`

  - **`watch_is_minimized(self, minimized: bool) -> None`:** Call `self._apply_minimized_state()`.

  - **`_apply_minimized_state(self) -> None`:**
    - Try to query `#terminal-tab-bar`, `#terminal-switcher`, `#terminal-status-line`
    - If minimized: set tab_bar.display = False, switcher.display = False, status_line.display = True, `self.styles.height = "auto"`
    - If not minimized: set tab_bar.display = True, switcher.display = True, status_line.display = False, `self.styles.height = TERMINAL_PANEL_HEIGHT`
    - Wrap in try/except for unmounted widget safety (same guard pattern as Phase 02-01 watch_show_hidden)

  - **`_update_tab_bar(self) -> None`:**
    - Get `switcher = self.query_one("#terminal-switcher", ContentSwitcher)`
    - Get `tab_bar = self.query_one("#terminal-tab-bar", Static)`
    - Build parts list: for each child in switcher.children, render `" [N] "` if active (bold/reverse style via Rich markup) or `"  N  "` if inactive, where N is 1-indexed position
    - Use Rich markup: active tab gets `"[bold reverse] {N} [/]"`, inactive gets `" {N} "`
    - Update tab_bar: `tab_bar.update(Text.from_markup(joined_parts))` using `from rich.text import Text`
    - Also update status line text: `TERMINAL_STATUS_RUNNING.format(count=len(children), s="s" if len(children) != 1 else "")`

  - **`_update_status_text(self) -> None`:**
    - Get switcher children count
    - Update `#terminal-status-line` with `TERMINAL_STATUS_RUNNING.format(count=count, s="s" if count != 1 else "")` if count > 0, else `TERMINAL_STATUS_IDLE`

  - **`on_pty_exited(self, message: PtyExited) -> None`:**
    - This handles shell exit within a tab (PtyExited bubbles up from TerminalWidget)
    - Find which tab's terminal posted this message by checking `message._sender`
    - Get the TerminalWidget that sent it: `terminal = message._sender` (if isinstance TerminalWidget)
    - Remove the terminal widget from the switcher
    - Mount a `Static` with the shell exited message in its place (using `SHELL_EXITED_MESSAGE.format(exit_code=message.exit_code)`) with `can_focus=True` and a unique id like `f"shell-exited-{terminal.id}"`
    - Actually, simpler approach: just show the exit message in the tab content area. Replace the dead terminal with a Static message. Create the Static, mount it in the switcher, set switcher.current to its id, remove the dead terminal.
    - Call `self._update_tab_bar()`

  - **`get_active_terminal(self) -> TerminalWidget | None`:**
    - Get switcher, check switcher.current
    - Try to get child by id, return if isinstance TerminalWidget, else None

  - **`stop_all_ptys(self) -> None`:**
    - Get switcher, iterate children, call `stop_pty()` on each TerminalWidget
    - Wrap each in try/except

  - **Property `tab_count(self) -> int`:** Return `len(list(self.query_one("#terminal-switcher", ContentSwitcher).children))` wrapped in try/except returning 0.
