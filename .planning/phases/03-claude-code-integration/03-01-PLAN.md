    **Create `nano_claude/terminal/pty_manager.py`:**
    This module manages the PTY subprocess lifecycle. It does NOT depend on Textual.

    Key components:
    - `KEY_MAP: dict[str, str]` -- Maps Textual key names to ANSI escape sequences. Include: arrow keys (up/down/left/right -> ESC[A/B/C/D), home (ESC[H), end (ESC[F), pageup (ESC[5~), pagedown (ESC[6~), delete (ESC[3~), enter (\r), backspace (\x7f), tab (\t), escape (\x1b), F1-F12 (standard xterm sequences).
    - `translate_key(event) -> str | None` -- Translates a Textual key event. Check KEY_MAP first, then ctrl+letter (chr(ord(letter.upper()) - 64)), then event.character for single printable chars. Return None for unrecognized.
    - `PYTE_COLOR_MAP: dict[str, str]` -- Maps pyte color names to Rich color names: black, red, green, brown->yellow, blue, magenta, cyan, white, default.
    - `render_pyte_screen(screen: pyte.Screen, cursor_visible: bool = True) -> list[Text]` -- Iterates `screen.buffer` row by row, char by char. For each `Char`, builds a `rich.style.Style` from fg/bg/bold/italics/underscore/reverse attributes using PYTE_COLOR_MAP. Appends `char.data` with style to a `Text()` line. If `cursor_visible` and row matches `screen.cursor.y`, applies "reverse" style at cursor.x position. Returns list of Text lines.
    - `class PtyManager` -- Manages one PTY subprocess:
      - `__init__(self)`: Sets `_pid = None`, `_fd = None`.
      - `spawn(self, command: str, cols: int, rows: int) -> tuple[int, int]`: Uses `pty.fork()`. Child process sets env TERM=xterm-256color, COLORTERM=truecolor, COLUMNS=str(cols), LINES=str(rows), then `os.execvpe(command, [command], env)`. Parent stores pid/fd, calls `fcntl.ioctl(fd, termios.TIOCSWINSZ, struct.pack("HHHH", rows, cols, 0, 0))`, returns (pid, fd).
      - `stop(self)`: If pid is set, sends SIGTERM. Polls `os.waitpid(pid, WNOHANG)` up to 10 times with 0.1s sleep. If still alive, sends SIGKILL and `os.waitpid(pid, 0)`. Always closes fd. Wraps everything in try/except (OSError, ChildProcessError). Resets pid and fd to None.
      - `resize(self, cols: int, rows: int)`: If fd is set, calls TIOCSWINSZ ioctl.
      - `is_running(self) -> bool`: If pid is None return False. Call `os.waitpid(pid, WNOHANG)` -- return True if result pid is 0, False otherwise. Catch ChildProcessError and return False.
      - `@property fd(self) -> int | None`: Returns _fd.
      - `@property pid(self) -> int | None`: Returns _pid.

    **Create `nano_claude/terminal/widget.py`:**
    The Textual widget that renders a PTY via pyte.

    Key components:
    - `class PtyDataReceived(Message)`: Contains `data: str`. Posted from read worker.
    - `class PtyExited(Message)`: Contains `exit_code: int | None`. Posted when PTY process ends.
    - `class TerminalWidget(Widget, can_focus=True)`:
      - `DEFAULT_CSS`: Set `overflow-y: auto; height: 1fr; width: 1fr;`
      - `__init__(self, command: str = "claude", **kwargs)`: Store command. Init `_screen: pyte.HistoryScreen | None = None`, `_stream: pyte.Stream | None = None`, `_pty_manager = PtyManager()`, `_running = False`. Use `pyte.HistoryScreen` with `history=10000` for scrollback per research open question 5.
      - `on_mount(self)`: Call `self.call_later(self._start_pty)` to defer until layout determines size.
      - `_start_pty(self)`: Get `cols = self.size.width`, `rows = self.size.height`. Create `self._screen = pyte.HistoryScreen(cols, rows, history=10000)`. Create `self._stream = pyte.Stream(self._screen)`. Connect stream to screen. Call `self._pty_manager.spawn(self._command, cols, rows)`. Set `self._running = True`. Call `self._start_read_loop()`.
      - `@work(thread=True, exclusive=True) _start_read_loop(self)`: While `self._running`, call `os.read(self._pty_manager.fd, 65536)`. If empty data, break. Decode with utf-8 errors=replace. Call `self.post_message(PtyDataReceived(decoded))` (thread-safe). On OSError break. After loop, call `self.app.call_from_thread(self._handle_pty_exit)`.
      - `on_pty_data_received(self, message: PtyDataReceived)`: Feed `message.data` to `self._stream`. Call `self.refresh()`.
      - `_handle_pty_exit(self)`: Set `self._running = False`. Post `PtyExited(exit_code=...)` -- get exit code from waitpid if possible.
      - `on_key(self, event: events.Key)`: If not running or no fd, return. Call `translate_key(event)`. If result is not None, write to PTY fd with `os.write(self._pty_manager.fd, char.encode("utf-8"))` and call `event.prevent_default()`. IMPORTANT: Do NOT prevent default for app-level bindings (ctrl+b, ctrl+e, ctrl+r, ctrl+q, ctrl+equal, ctrl+minus, ctrl+backslash, ctrl+h, ctrl+s, ctrl+f) -- let those bubble to the app. Check by comparing event.key against a frozenset of reserved keys.
      - `on_resize(self, event: events.Resize)`: If running, update pyte screen with `self._screen.resize(event.size.height, event.size.width)` and call `self._pty_manager.resize(event.size.width, event.size.height)`.
      - `render(self) -> RenderResult`: If screen is None, return `Text("Starting Claude Code...")`. Call `render_pyte_screen(self._screen)`. Join lines with newline. Return the joined Text.
      - `stop_pty(self)`: Set `self._running = False`. Call `self._pty_manager.stop()`.
      - `restart_pty(self)`: Call `stop_pty()`. Call `self.call_later(self._start_pty)`.
